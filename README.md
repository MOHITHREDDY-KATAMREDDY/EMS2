\# Employee Management System (EMS)



\## 📌 Project Description

This is a simple Spring Boot application to manage employee data using REST APIs and MySQL database.



\---



\## 🛠️ Tech Stack

\- Java

\- Spring Boot

\- Spring Data JPA (Hibernate)

\- MySQL

\- Maven



\---



\## 🚀 Features

\- Add Employee

\- Get All Employees

\- Get Employee By ID

\- Delete Employee



\---



\## ⚙️ Setup \& Run



1\. Clone the repository:

&#x20;  git clone https://github.com/YOUR-USERNAME/EMS2.git



2\. Open project in STS / IntelliJ



3\. Configure database in application.properties:

&#x20;  spring.datasource.url=jdbc:mysql://localhost:3306/ems

&#x20;  spring.datasource.username=root

&#x20;  spring.datasource.password=your\_password



4\. Run the application:

&#x20;  Run EmsApplication.java
